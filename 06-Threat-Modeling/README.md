# Threat Modeling

This project demonstrates a basic threat modeling exercise for a web application.

## Scope
- Web application with internet access
- Database containing user credentials

## Identified Threats
- Phishing and credential theft
- Unauthorized access to user accounts
- Denial of Service (DoS) attacks

## Risk Level
High impact due to potential data breaches and service disruption.

## Mitigations
- Multi-Factor Authentication (MFA)
- Security awareness training
- Traffic monitoring and filtering
