# Risk Assessment & Security Audit

## Project Overview
This project presents a comprehensive internal IT security audit conducted for a fictional organization, Botium Toys. The assessment evaluates the organization’s security posture, identifies risks to critical assets, and reviews compliance with key regulatory standards.

The audit focuses on identifying gaps in technical, administrative, and operational controls and provides a risk-based analysis aligned with real-world security and compliance practices.

## Audit Scope
The scope of this assessment includes:
- Employee and IT equipment
- Internal network infrastructure
- Business systems and applications
- Data storage and retention practices
- Legacy systems and operational processes

## Objectives
- Identify security risks and control gaps
- Assess compliance with regulatory requirements
- Evaluate asset management and access controls
- Determine the organization’s overall security risk level

## Standards & Frameworks Referenced
- PCI DSS (Payment Card Industry Data Security Standard)
- GDPR (General Data Protection Regulation)
- SOC (System and Organization Controls)

## Key Outcomes
- Identified critical gaps in access control, encryption, monitoring, and disaster recovery
- Determined a high overall security risk level
- Highlighted compliance deficiencies that could lead to regulatory penalties
- Provided prioritized remediation recommendations

## Skills Demonstrated
- Risk assessment and audit documentation
- Security control evaluation
- Regulatory compliance analysis (PCI DSS, GDPR, SOC)
- Asset identification and classification
- Risk-based security reporting
