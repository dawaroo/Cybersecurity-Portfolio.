# Investigation Evidence & Notes

This investigation focuses on the installation and management of security monitoring tools on a Linux system.

## Risks Addressed
Improper package management can lead to outdated or misconfigured security tools, reducing visibility and detection capabilities. Failing to install or maintain tools such as IDS/IPS or packet capture utilities may allow malicious activity to go undetected.

## Mitigation
Using a reliable package manager ensures that security tools are installed correctly, updated regularly, and maintained throughout their lifecycle, supporting effective monitoring and incident response.
