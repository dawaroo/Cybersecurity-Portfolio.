# Package Management and Security Tools

## Project Description
This project demonstrates how Linux package management is used to install, remove, and maintain security monitoring tools on a Linux system.

The objective of this investigation is to show how proper package management supports security operations by ensuring that critical tools such as intrusion detection systems and packet capture utilities are correctly installed, updated, and maintained.

## Tools Installed
- Suricata (IDS/IPS)
- tcpdump (packet capture and traffic analysis)

## Investigation Objective
- Verify the availability of the package manager
- Install and remove security tools using APT
- Reinstall tools to ensure integrity and proper configuration
- Demonstrate safe and controlled package management practices
