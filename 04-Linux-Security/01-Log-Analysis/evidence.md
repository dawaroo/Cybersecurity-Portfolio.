# Investigation Evidence & Notes

## Log Analysis Observations
- Multiple authentication failures were detected in server_logs.txt
- Repeated "Unauthorized access" entries indicate possible credential misuse or brute-force attempts
- System warnings related to disk usage may impact availability if not addressed

## User Data Findings
- Deleted and newly added users were identified through report files
- Department-level user changes were successfully traced using pattern matching

## Analyst Notes
This investigation demonstrates how basic Linux tools can be leveraged to support security monitoring, incident detection, and forensic-style data analysis without relying on advanced SIEM platforms.
